품꿈힘 한자어 공부 v2.1 (모바일/패드 실행 가이드)

1) 가장 쉬운 실행(스마트폰/패드에서 '그냥 열기'로 실행)
- 일부 기기(iOS 파일앱/Safari 등)는 file://로 열 때 외부 JS 로딩을 차단합니다.
- 이럴 때는 'index_singlefile.html'을 여세요. (단일 파일이라 차단 없이 동작)

2) PWA(홈화면 추가/오프라인 캐시)로 쓰려면
- index.html / manifest.json / sw.js 를 https 웹서버에 올려야 합니다.
  (GitHub Pages, 학교 홈페이지, NAS, 간단한 정적 호스팅 등)
- 그런 다음 브라우저에서 접속 → '홈 화면에 추가'로 설치

3) PC에서 로컬 웹서버로 실행(권장)
- 이 폴더에서 터미널 실행 후:
  python -m http.server 8000
- 브라우저에서:
  http://localhost:8000/index.html

문제가 계속되면
- 브라우저 개발자 도구(Console) 오류 메시지를 알려주시면, 그 줄 기준으로 즉시 잡아드릴게요.
